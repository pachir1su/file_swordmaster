# file_swordmaster

Python을 활용하여 오디오 파일과 PDF 파일을 간편하게 분할하는 스크립트 모음입니다. 로컬 환경은 물론 Google Colab에서도 직관적인 **웹 UI(Gradio)**를 통해 클릭 몇 번으로 실행할 수 있도록 변경되었습니다.


## 제공하는 기능


### 1. Web UI Dashboard (통합 대시보드)
* `Gradio` 라이브러리를 활용하여 터미널 명령어 없이 브라우저 환경에서 작동합니다.
* 파일 업로드, 옵션 설정, 결과물 다운로드를 드래그 앤 드롭과 마우스 클릭만으로 처리할 수 있습니다.
* 탭(Tab) 인터페이스를 통해 PDF와 오디오 도구를 손쉽게 전환합니다.

### 2. Audio Swordmaster (오디오 분할/편집기)
* `pydub` 라이브러리 기반
* **정밀 타겟팅 분할:** 단순 초(sec) 단위뿐만 아니라 `2:04`, `1:02:04`와 같은 직관적인 '분:초' 시간 포맷을 자동으로 인식하여 원하는 구간만 정확하게 잘라냅니다.
* **무음 베어내기 (Auto-trimming):** `split_on_silence` 모듈을 활용하여, 강의나 회의 녹음 파일 등에서 소리가 나지 않는 무음 구간을 자동으로 감지하고 제거합니다.


### 3. PDF Swordmaster (PDF 분할기)
* `pypdf` 라이브러리 기반
* **다중 범위 정밀 추출:** 단순한 '시작~끝' 페이지 추출을 넘어, `1, 3, 5-10`과 같이 콤마(-)와 하이픈(-)을 섞어 불연속적인 페이지들을 입력해도 해당 페이지만 정확히 뽑아 하나의 PDF로 병합해 줍니다.
