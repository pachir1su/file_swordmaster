# file_swordmaster

Python을 활용하여 오디오 파일과 PDF 파일을 간편하게 분할하는 스크립트 모음입니다. Google Colab 및 로컬 환경에서 실행할 수 있도록 작성되었습니다.

## 제공하는 기능

* **Audio Splitter**
  * `pydub` 라이브러리를 사용합니다.
  * 업로드된 `.mp3`, `.m4a` 등의 음성 파일을 자동으로 인식하여 원하는 개수(N등분)로 분할하고 저장합니다.
* **PDF Splitter**
  * `pypdf` 라이브러리를 사용합니다.
  * 원본 PDF 파일에서 사용자가 지정한 특정 페이지 범위만 추출하여 새로운 PDF 파일로 분리합니다.
