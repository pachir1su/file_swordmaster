# file_swordmaster

Python을 활용하여 오디오 파일과 PDF 파일을 간편하게 분할하는 스크립트 모음입니다. 로컬 환경은 물론 Google Colab에서도 직관적인 웹 UI를 통해 클릭 몇 번으로 실행할 수 있습니다.

👉 **[웹에서 바로 실행하기 (Hugging Face Space)](https://huggingface.co/spaces/capybaranice/file_swordmaster)**

---

## 제공하는 기능

### 1. ALL_swordmaster (통합 대시보드)
* `Gradio` 라이브러리를 활용하여 터미널 명령어 없이 브라우저 환경에서 작동합니다.
* 파일 업로드, 옵션 설정, 결과물 다운로드를 드래그 앤 드롭과 마우스 클릭만으로 처리할 수 있습니다.
* 탭 인터페이스를 통해 모든 도구를 손쉽게 전환하며 사용할 수 있습니다.

### 2. 오디오 분할/편집기
* `pydub` 라이브러리 기반
* **정밀 분할 :** 단순 초(sec) 단위 뿐만 아니라 `2:04`, `1:02:04`와 같은 직관적인 '분:초' 시간 포맷을 자동으로 인식하여 원하는 구간만 정확하게 잘라냅니다.
* **무음 제거 :** `split_on_silence` 모듈을 활용해 소리가 나지 않는 무음 구간을 자동으로 감지하고 제거합니다.

### 3. PDF 분할기
* `pypdf` 라이브러리 기반
* **다중 범위 정밀 추출 :** '시작~끝' 페이지 추출 뿐만 아니라 `1, 3, 5-10`과 같이 콤마(,)와 하이픈(-)을 섞어 불연속적인 페이지들을 입력해도 해당 페이지만 정확히 뽑아 하나의 PDF로 분리해 줍니다.

### 4. PDF 병합 및 잠금 해제
* `pypdf` 라이브러리 기반
* **PDF 병합 :** 여러 개의 PDF 파일을 업로드하면 올린 순서대로 하나의 거대한 PDF 파일로 합쳐줍니다.
* **비밀번호 해제 :** 읽기 전용이나 암호가 걸려있는 PDF 파일의 경우 비밀번호를 한 번만 입력해 주면 암호가 완전히 해제된 깨끗한 복사본을 만들어냅니다.

### 5. 상 음원 추출기
* `moviepy` 라이브러리 기반
* **소리 추출 :** `.mp4`, `.avi` 등의 비디오 파일을 업로드하면 영상은 제외하고 오디오 트랙만 `.mp3` 파일로 빠르게 추출하여 저장해 줍니다.

### 6. PDF 잠금 및 워터마크
* `pypdf`, `reportlab` 라이브러리 기반
* **문서 암호화 :** 누구나 열어볼 수 있는 PDF 파일에 새로운 비밀번호를 설정하여 잠가버립니다.
* **텍스트 워터마크 :** 원하는 텍스트(예: 대외비, 기밀문서 등)를 입력하면 문서 정중앙에 대각선으로 투명한 워터마크를 찍어 문서의 무단 도용을 방지합니다. (한글 개싹싹 지원)
