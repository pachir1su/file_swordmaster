# file_swordmaster

PDF와 오디오 파일을 편집하는 스크립트 입니다. 로컬 환경은 물론 Google Colab에서도 실행할 수 있습니다.

👉 **[웹에서 바로 실행하기 (Hugging Face Space)](https://huggingface.co/spaces/capybaranice/file_swordmaster)**

---

## 제공하는 기능

### PDF
- **분할**
- **병합**
- **비밀번호 해제**
- **비밀번호 설정**
- **워터마크 설정**
- **텍스트 -> PDF 변환**


### 오디오
- **오디오 자르기**
- **N분할**
